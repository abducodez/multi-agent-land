# Demo Runbook

## Opening Seed

Use:

```text
A village of stage props wakes up and argues about which fairy tale they belong to.
```

## Demo Beat

1. Start the run.
2. Advance two turns to show agent specialization.
3. Inject: `A lantern starts whispering recipes for impossible soup.`
4. Advance again and point to the ledger.
5. Explain that every visible state is derived from append-only events.

## What To Emphasize

- This is not a chat wrapper.
- The agents have roles and produce the experience.
- The UI exposes how the tiny system thinks.
- The project is built to support <=32B and <=4B model modes.
